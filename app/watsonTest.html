Drop Down

{{dropDownClicked}}

Drop Down Options

  • title <optional>: The name that appears on the drop down button
  • drop-down-icon <optional>: The icon that appears to the right of the title
  • myClass <optional>: Additional classes to add to the drop down button
  • action <optional> function to call when a menu item is clicked
  • menuItems: list of items with the structure - item {name, icon}

Search

{{searchClicked}}

Search Options

  • place-holder-text <optional>: The place holder text
  • button-text <optional>: The text that appears on the button
  • action: The function to call when the button is pressed
  • button-icon <optional> The icon that appears on the button
  • search-text: The variable that the text from the input box is stored in
  • button-class: <optional> Additional classes to add to the button
  • text-class: <optional> Additional classes to add to the input box

File Upload

File name: {{file.details.name}}

File Content: {{file.data}}

File Upload Options

  • large <optional>: If set to true will show the large version of the file upload
  • format <optional>: The format the file should be read in. Either "text", "url", or "binary"
  • file: The variable that contains the uploaded file. "file.details" contains the meta data of the file e.g. "file.details.name". "file.data" contains the contents of this file

Modal Options

  • type <optional>: The type of modal to display set to "success", "error", "warning", "default"
  • title: The title that should be shown on the modal
  • id: The id for the modal (needed to be able to open it)
  • action <optional>: The function to call when the modal is closed
  • okTitle: The text to appear on the ok button
  • cancelTitle <optional>: The function to call when the modal is closed
  • disableOkExpression <optional>: The expression to determine if the ok button should be disabled
  • hideOkExpression <optional>: The expression to determine if the ok button should be hidden

Loading

  • loadingMessage <optional>: The message to display while loading

Alerts

Alerts Options

  • level: The level of the alert 'error', 'warning', or 'info'
  • dismissable: Whether the alert can be closed or not
  • title: The title of the alert
  • text: The text the laert should have
  • link <optional>: a link to more information
  • link text <optional>: the text to be displayed on the link

Alerts can also be added with a service. In your controller add watsonAlerts to the dependencies. Then use the methods add, remove, and clear.

Toggle Options

  • value: true==yes, false==no

Footer

Footer Options

  • translations <optional>: a literal object map representing the link-texts as properties with corresponding translation strings, example usage below (in German),
    • contact: 'Kontakt'
    • privacy: 'Privatsphäre'
    • terms: 'Nutzungsbedingungen'
