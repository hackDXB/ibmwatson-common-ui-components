Drop Down

{{dropDownClicked}}

Drop Down Options

  • title <optional>: The name that appears on the drop down button
  • drop-down-icon <optional>: The icon that appears to the right of the title
  • myClass <optional>: Additional classes to add to the drop down button
  • action <optional> function to call when a menu item is clicked
  • menuItems: list of items with the structure - item {name, icon}

Search

{{searchClicked}}

Search Options

  • place-holder-text <optional>: The place holder text
  • button-text <optional>: The text that appears on the button
  • action: The function to call when the button is pressed
  • button-icon <optional> The icon that appears on the button
  • search-text: The variable that the text from the input box is stored in
  • button-class: <optional> Additional classes to add to the button
  • text-class: <optional> Additional classes to add to the input box

File Upload

File name: {{file.details.name}}

File Content: {{file.data}}

File Upload Options

  • large <optional>: If set to true will show the large version of the file upload
  • format <optional>: The format the file should be read in. Either "text", "url", or "binary"
  • file: The variable that contains the uploaded file. "file.details" contains the meta data of the file e.g. "file.details.name". "file.data" contains the contents of this file
